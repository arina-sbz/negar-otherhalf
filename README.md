# negar-otherhalf

In this repository I keep the surprises I make for Negar.

## The Little Universe of My Other Half

A handmade, scrapbook-style birthday website — a guided emotional experience of memories, letters, a friendship timeline, a friendship quiz, a Taylor Swift "Swiftie corner," a playlist, and a custom Moon-and-Saturn constellation, ending in a final birthday reveal.

Built with Vite, React, TypeScript, Tailwind CSS, Motion, and canvas-confetti — fully static, no backend required.

### Quick start

```bash
npm install
npm run dev
```

### Personalizing

Everything editable lives in one file: [`src/data/birthdayContent.ts`](src/data/birthdayContent.ts). See [`PERSONALIZATION_GUIDE.md`](PERSONALIZATION_GUIDE.md) for a full walkthrough, and [`DEPLOYMENT_GUIDE.md`](DEPLOYMENT_GUIDE.md) for deploying to Vercel.
